The shorthening logic is performed on the side of Client. The 'generateShortUrl' function accepts a longUrl as input and returns a shortened URL. It employs a custom base-62 encoding to generate an 8-character hash. This hash is formed by randomly selecting characters from the base-62 set. Next, the function extracts the domain from the longUrl and constructs the shortened URL by appending the hash to the protocol and domain. The 'getRandomInt' function is a helper used to generate random integers within a given range for selecting characters from the base-62 set. The resulting shortened URL follows the format 'protocol://domain/hash', providing a concise representation of the original longUrl.
